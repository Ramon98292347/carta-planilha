# Painel de Gestão

Sistema para gestão de cartas e obreiros com React + Vite + TypeScript.

## Requisitos

- Node.js 18+
- npm

## Rodando localmente

```bash
npm install
npm run dev
```

Aplicação em desenvolvimento: `http://127.0.0.1:5173/` (ou porta exibida no terminal).

## Build de produção

```bash
npm run build
npm run preview
```

## PWA

O projeto está configurado como PWA:
- Manifest Web App
- Service Worker com atualização automática
- Ícone do app em `public/app-icon.svg`
